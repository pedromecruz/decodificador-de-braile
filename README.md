## Decodificador de ASCII(Binário) para Braille - Representação lógica

Neste nanoprojeto, a intenção foi iniciar um mecanismo de acessibilidade. A função do circuito lógico é transformar um caractere ASCII, representado por binário, em Braille. 

Atualmente, o projeto comporta A-Z maiúsculos e minúsculos, 0-9 e os acentos usados no português brasileiro, maíusculos e minúsculos. 

Todos os caracteres suportados e representados foram baseados na documentação oficial de Braille brasileiro e da representação oficial de ASCII para binário.

Para o desenvolvimento do projeto, foi usado o Logisim, que deixei upado no git também para testes!
