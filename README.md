# Decodificador de ASCII para Braille

## Conceitos do projeto
O decodificador tem como objetivo permitir e viabilizar a leitura tátil de telas de computador, sendo apenas a parte final do sistema. 
Assim, através de um programa que interpreta e faz a leitura da tela, nosso decodificador fará todo o trabalho de levantamento de pinos para um aparelho físico, que trocará letra a letra em braille, possibilitando a leitura de toda uma tela de computador sem depender de um leitor sonoro de telas.
